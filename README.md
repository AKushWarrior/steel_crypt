A simple library of high-level API's and sugar for PointyCastle. This 
library currently supports both hashing and two-way encryption:

#### 2-Way
* AES ('AES') _(Default)_
* Salsa20 ('Salsa20')
* More coming...

#### Hashing
* SHA-3 ('SHA-3)
* Blake2B ('Blake2b')
* MD5 ('MD5')
* SHA-1 ('SHA-1')
* SHA-512 ('SHA-512)
* SHA-256 ('SHA-256')
* Whirlpool ('Whirlpool)
